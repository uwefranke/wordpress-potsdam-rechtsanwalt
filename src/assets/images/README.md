# Hero-Bild Platzhalter

Dieses Verzeichnis ist für das Hero-Hintergrundbild vorgesehen.

## Erforderliches Bild

**Dateiname**: `potsdam-skyline.jpg`

### Spezifikationen:
- **Format**: JPG oder PNG
- **Größe**: 1920 x 500 Pixel (empfohlen)
- **Motiv**: Potsdamer Skyline mit Nikolaikirche
- **Qualität**: Hochauflösend, leicht unscharf für bessere Textlesbarkeit
- **Dateigröße**: Möglichst optimiert (unter 500KB)

### So erhalten Sie ein passendes Bild:

1. **Eigenes Foto**
   - Fotografieren Sie die Potsdamer Skyline
   - Bearbeiten Sie das Bild (Größe, Weichzeichner)

2. **Stock-Fotos** (mit Lizenz)
   - Adobe Stock
   - Shutterstock
   - Unsplash (kostenlos)
   - Pixabay (kostenlos)

3. **Suchbegriffe**
   - "Potsdam Skyline"
   - "Potsdam Nikolaikirche"
   - "Potsdam Stadtbild"

### Alternative: Customizer verwenden

Anstatt die Datei hier abzulegen, können Sie das Bild auch über den WordPress Customizer hochladen:
1. Design → Customizer
2. Hero-Bereich
3. Hero Hintergrundbild → Bild auswählen

---

**Hinweis**: Achten Sie auf die Bildrechte! Verwenden Sie nur lizenzfreie oder eigene Bilder.
