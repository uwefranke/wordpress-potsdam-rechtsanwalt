# Screenshot Info

## Theme-Screenshot erstellen

Für die WordPress-Theme-Übersicht benötigen Sie ein Screenshot-Bild:

### Anforderungen
- **Dateiname**: `screenshot.png`
- **Größe**: 1200x900 Pixel (Verhältnis 4:3)
- **Format**: PNG
- **Speicherort**: Root-Verzeichnis des Themes

### So erstellen Sie den Screenshot:

1. **Option 1: Browser-Screenshot**
   - Aktivieren Sie das Theme in WordPress
   - Öffnen Sie die Startseite in Ihrem Browser
   - Erstellen Sie einen Screenshot der Seite
   - Schneiden Sie das Bild auf 1200x900 Pixel zu
   - Speichern Sie es als `screenshot.png` im Theme-Verzeichnis

2. **Option 2: Design-Tool**
   - Verwenden Sie Photoshop, GIMP oder Canva
   - Erstellen Sie ein neues Dokument (1200x900px)
   - Fügen Sie einen Screenshot Ihrer Website ein
   - Exportieren Sie als PNG

3. **Option 3: Online-Tool**
   - Nutzen Sie Websites wie ResizeImage.net
   - Laden Sie Ihren Screenshot hoch
   - Passen Sie die Größe auf 1200x900px an
   - Laden Sie die PNG-Datei herunter

### Was sollte auf dem Screenshot zu sehen sein?
- Der Header mit Logo und Navigation
- Ein Teil des Hero-Bereichs
- Die Service-Karten (teilweise)
- Die Sidebar mit Kontaktformular

Dies gibt potenziellen Nutzern einen guten Eindruck vom Theme-Design.

---

**Hinweis**: Ohne screenshot.png wird in WordPress ein Platzhalter angezeigt. 
Das Theme funktioniert auch ohne Screenshot, aber mit Screenshot sieht es professioneller aus.
