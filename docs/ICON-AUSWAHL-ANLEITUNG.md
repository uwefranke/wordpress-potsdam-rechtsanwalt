# Icon-Auswahl für Rechtsgebiete - Anleitung

Mit dem Potsdam Rechtsanwalt Theme können Sie für jedes Rechtsgebiet auf der Startseite ein individuelles Icon auswählen. Dieses Feature ermöglicht eine visuelle Differenzierung Ihrer Dienstleistungen.

## 📍 Wo finde ich die Icon-Auswahl?

1. **WordPress Admin** einloggen
2. **Design → Customizer** öffnen
3. **Rechtsgebiete** auswählen
4. Bei jedem Rechtsgebiet (1-8) finden Sie die Option **"Icon"**

## 🔧 So ändern Sie die Icons

### Schritt-für-Schritt-Anleitung

1. **Customizer öffnen**
   - WordPress Admin → Design → Customizer

2. **Rechtsgebiete auswählen**
   - Im linken Menü auf "Rechtsgebiete" klicken

3. **Icon auswählen**
   - Bei jedem Rechtsgebiet finden Sie:
     - **Titel** (z.B. "Mietrecht")
     - **Icon** ← Hier das gewünschte Icon auswählen
     - **Beschreibung**
     - **Link**

4. **Icon aus Dropdown wählen**
   - Klicken Sie auf das Icon-Dropdown
   - Wählen Sie das passende Icon aus
   - Die Icons haben Emoji-Vorschau und deutsche Beschreibung

5. **Live-Vorschau**
   - Das neue Icon wird sofort in der Vorschau angezeigt
   - Prüfen Sie das Ergebnis auf der rechten Seite

6. **Veröffentlichen**
   - Klicken Sie auf **"Veröffentlichen"** oben links
   - Die Änderungen sind sofort live

## 🎨 Design-Richtlinien

### Konsistenz wahren

- Verwenden Sie nicht mehr als 4-6 verschiedene Icons auf der Startseite
- Icons sollten thematisch zusammenpassen
- Die **Waage** (⚖️ scale) ist das universelle Rechtssymbol

### Visuelles Gleichgewicht

- Mischen Sie nicht zu viele ähnliche Icons (z.B. nur Gebäude-Icons)
- Wechseln Sie zwischen geometrischen und organischen Formen
- Vermeiden Sie zu viele Icons mit vielen Details

### Beispiel-Kombinationen

**Option 1: Klassisch (Rechts-fokussiert)**
1. ⚖️ scale - Allgemeines Recht
2. 🔨 gavel - Gerichtsverfahren
3. 📋 file-text - Vertragsrecht
4. 🤝 handshake - Mediation

**Option 2: Themen-spezifisch**
1. 🏠 home - Immobilienrecht
2. 🚗 car - Verkehrsrecht
3. 👥 users - Familienrecht
4. 💼 briefcase - Arbeitsrecht

**Option 3: Gemischt**
1. ⚖️ scale - Mietrecht
2. 🏠 home - Immobilienrecht
3. 🔧 tool - Baurecht
4. 🛡️ shield - Versicherungsrecht

## 🔧 Technische Details

### Icon-System

- **Quelle**: Feather Icons (Open Source, MIT License)
- **Format**: SVG (Scalable Vector Graphics)
- **Größe**: 32×32 Pixel (skaliert automatisch)
- **Farbe**: Passt sich automatisch dem Theme an (`currentColor`)
- **Performance**: Inline SVG, kein externes Laden

### Anpassung per CSS (für Entwickler)

```css
/* Icon-Farbe ändern */
.service-icon svg {
    color: #1a3a5c; /* Marineblau */
}

/* Icon-Größe anpassen */
.service-icon svg {
    width: 40px;
    height: 40px;
}

/* Icon-Hintergrund */
.service-icon {
    background: #f5e6d3;
    border-radius: 50%;
    padding: 15px;
}
```

## ❓ Häufige Fragen (FAQ)

### Kann ich eigene Icons hochladen?

Aktuell nicht über den Customizer. Die 20 vordefinierten Icons decken die meisten Rechtsgebiete ab. Für Custom-Icons kontaktieren Sie Ihren Entwickler.

### Werden die Icons auf mobilen Geräten angezeigt?

Ja, die Icons sind vollständig responsive und werden auf allen Geräten korrekt dargestellt.

### Was passiert, wenn ich kein Icon auswähle?

Das System verwendet automatisch das Icon **📋 file-text** (Vertrag) als Fallback.

### Kann ich die Icon-Reihenfolge ändern?

Die Icons erscheinen in der Reihenfolge der Rechtsgebiete (1-8). Um die Reihenfolge zu ändern, tauschen Sie einfach die Inhalte (Titel, Beschreibung, Link, Icon) zwischen den Rechtsgebieten.

### Wie viele Rechtsgebiete kann ich anzeigen?

Sie können 1-8 Rechtsgebiete anzeigen. Die Anzahl stellen Sie im Customizer unter **"Anzahl Rechtsgebiete"** ein.

### Werden die Icons im Dark Mode angezeigt?

Ja, die Icons passen sich automatisch an den Dark Mode an und bleiben gut lesbar.

## 🐛 Problemlösung

### Icon wird nicht angezeigt

1. **Browser-Cache leeren**: Strg+F5 (Windows) / Cmd+Shift+R (Mac)
2. **Customizer-Einstellungen prüfen**: Ist das richtige Icon ausgewählt?
3. **WordPress-Cache leeren**: Falls Sie ein Caching-Plugin verwenden

### Icon sieht anders aus als in der Vorschau

- **Browser-Kompatibilität**: SVG-Icons werden von allen modernen Browsern unterstützt
- **Theme-Update**: Stellen Sie sicher, dass Sie die neueste Version verwenden

### Änderungen werden nicht gespeichert

1. Klicken Sie auf **"Veröffentlichen"** oben links im Customizer
2. Warten Sie auf die Bestätigung "Veröffentlicht"
3. Laden Sie die Startseite neu (Strg+F5)
